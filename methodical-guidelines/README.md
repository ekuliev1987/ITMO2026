## Общая информация

Методические указания пишутся в свободной форме, исходя из личного опыта и мироощущения авторов. Главное — не формальность, а:
- (а) удобство использования другими людьми
- (б) отсутствие «дыр» и ошибок

## Формат занятий

- **Лекции:** информативные, с акцентом на ключевые концепции
- **Практические занятия:** hands-on работа с кодом, обязательна защита отчётов
- **Самостоятельная работа:** подготовка к занятиям, выполнение продвинутых заданий

## Оценивание

**Примечание:** Приведённые ниже баллы являются примерными. Преподаватель может гибко изменять распределение баллов в зависимости от сложности работы и уровня освоения компетенций.

| Элемент | Макс. баллы |
|---------|-------------|
| Практические занятия (M1-M6) | до 200 |
| Контрольная работа | до 40 |
| Проект (опционально) | до 100 |
| Экзамен | до 40 |
| **Итого** | **до 380** |
