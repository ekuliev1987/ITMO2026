# Рекомендации преподавателям по оцениванию

## Система оценивания

Выбор количества видов КИМ для конкретной РПД, а также их сочетаний полностью остаётся на совести разработчиков ФОС.

## Текущий контроль

**Примечание:** Выбор количества видов КИМ и их сочетаний полностью остаётся на совести разработчиков ФОС. Приведённые баллы являются примерными.

Виды контроля определяются преподавателем исходя из:
- Объёма и сложности материала
- Требуемого уровня освоения компетенций
- Доступных ресурсов

Возможные формы:
- Защита практических работ
- Тестирование
- Устный опрос
- Проектные работы

## Рубежный контроль

Проводится по завершении модуля:
- Защита практических работ
- Устный опрос по теории
- Проверка понимания ключевых концепций

## Итоговая аттестация

**Экзамен:** 2 вопроса, 30-40 минут на ответ

| Баллы | Оценка | Описание |
|-------|--------|----------|
| 35-40 | Отлично | Полный верный ответ, концептуальное понимание, примеры |
| 29-34 | Хорошо | Достаточно полный ответ, небольшие неточности |
| 22-28 | Удовлетворительно | Неполный ответ, трудности при обобщении |
| 0-21 | Неудовлетворительно | Разрозненные знания, ошибки |

## Критерии оценивания практических работ

### Код (50%)
- Корректность работы
- Читаемость и стиль
- Использование лучших практик

### Отчёт (30%)
- Описание цели и хода выполнения
- Анализ результатов
- Выводы

### Защита (20%)
- Понимание выполненного
- Ответы на вопросы
- Умение объяснить решения

## Работа с академической честностью

1. **Код:** проверяйте уникальность решений, задавайте уточняющие вопросы
2. **Отчёты:** требуйте личных выводов и анализа
3. **Защита:** задавайте вопросы по деталям реализации

## Ликвидация задолженностей

Студенты, не набравшие необходимый минимум, могут ликвидировать задолженность:
- До конца последней недели теоретического обучения
- По согласованию с преподавателем
- С выполнением дополнительных заданий
