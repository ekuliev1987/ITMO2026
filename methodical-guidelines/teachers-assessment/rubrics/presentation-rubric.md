# Критерии оценки защиты практической работы

## Шаблон рубрикатора

**Примечание:** Этот рубрикатор является шаблоном. Конкретные веса баллов и пороговые значения требуют экспертной валидации после первого запуска курса.

---

## 1. Структура презентации (20%)

| Критерий | 0 баллов | 1 балл | 2 балла |
|----------|----------|--------|---------|
| Введение | Нет цели и задачи | Цель указана, но неясна | Чёткая цель, актуальность, план |
| Теоретическая часть | Отсутствует | Минимальная, без ссылок | Полная, с источниками и формулами |
| Практическая часть | Нет демонстрации | Код показан, без объяснений | Код + пояснения + результаты |
| Выводы | Отсутствуют | Формальные | Конкретные, с анализом результатов |

---

## 2. Качество презентации (20%)

| Критерий | 0 баллов | 1 балл | 2 балла |
|----------|----------|--------|---------|
| Визуальное оформление | Хаотично, перегружено | Есть структура, есть недочёты | Лаконично, визуально понятно |
| Слайды | Текст вместо тезисов | Смесь текста и схем | Схема/график + краткие тезисы |
| Читаемость | Шрифт мелкий, контраст низкий | Читаемо, но есть проблемы | Чёткий шрифт, хороший контраст |

---

## 3. Устное выступление (30%)

| Критерий | 0 баллов | 1 балл | 2 балла |
|----------|----------|--------|---------|
| Время | <50% или >120% нормы | 75-100% нормы | 90-100% нормы |
| Глубина понимания | Чтение со слайдов | Чтение + пояснения | Свободное изложение, без чтения |
| Логика изложения | Неясная структура | Есть структура, пропуски | Чёткая логика, переходы |
| Умение выделять главное | Все детали равнозначны | Есть акценты | Ключевые идеи выделены |

---

## 4. Ответы на вопросы (30%)

| Критерий | 0 баллов | 1 балл | 2 балла |
|----------|----------|--------|---------|
| Понимание вопроса | Не понял вопрос | Понял, но затрудняется | Чётко понял, уточнил при необходимости |
| Полнота ответа | Не отвечает или не по теме | Частичный ответ | Полный, аргументированный ответ |
| Обоснование решений | Нет обоснования | Есть обоснование, но слабое | Чёткое обоснование с примерами |
| Работа с критикой | Защищается, спорит | Принимает замечания | Конструктивно, предлагает решения |

---

## Итоговая оценка

| Сумма баллов | Оценка |
|--------------|--------|
| 36-40 | Отлично |
| 29-35 | Хорошо |
| 20-28 | Удовлетворительно |
| 0-19 | Неудовлетворительно |

## Пример заполнения

| Критерий | Баллы | Комментарий |
|----------|-------|-------------|
| **Структура презентации** | **7/8** | Отсутствовал план работы |
| **Качество презентации** | **4/4** | Отличный визуальный стиль |
| **Устное выступление** | **5/6** | Чуть превысил время (+2 мин) |
| **Ответы на вопросы** | **6/6** | Полные ответы, хорошая аргументация |
| **ИТОГО** | **22/24** | **Хорошо** |

## Заметки для преподавателя

- [ ] Указать нормативное время выступления (например, 7 минут)
- [ ] Определить допустимое отклонение по времени
- [ ] При необходимости добавить критерий «Уникальность решения»
- [ ] Уточнить веса критериев после анализа результатов
