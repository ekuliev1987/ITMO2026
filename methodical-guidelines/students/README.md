# Рекомендации обучающимся

## Как учиться эффективно

### Перед занятием
1. Просмотрите тему модуля в README
2. Повторите базовые понятия из предыдущих модулей
3. Подготовьте окружение (Google Colab или локальная среда)

### Во время занятия
1. Записывайте ключевые концепции
2. Не копируйте код слепо — пытайтесь понимать каждую строку
3. Задавайте вопросы по ходу занятия

### После занятия
1. Оформите отчёт по практическому заданию
2. Протестируйте код на дополнительных примерах
3. Повторите теорию по конспекту

## Советы по модулям

### Модуль 1 (Введение в ML)
- Освежите знания Python: списки, циклы, функции
- Разберитесь с основами NumPy: массивы, индексация, broadcasting
- Практикуйтесь в Pandas: DataFrame, группировка, агрегация

### Модуль 2 (Автоэнкодеры и VAE)
- Повторите основы нейронных сетей: прямой/обратный проход
- Поймите разницу между AE и VAE
- Экспериментируйте с гиперпараметрами

### Модуль 3 (GAN)
- GAN сложны в обучении — будьте готовы к проблемам
- Изучите техники стабилизации: label smoothing, gradient penalty
- Используйте предобученные модели для начала

### Модуль 4 (Трансформеры и LLM)
- Разберитесь с механизмом Attention — это ключ к пониманию
- Практикуйте Prompt Engineering на разных задачах
- Начинайте с малых моделей (DistilBERT, TinyLlama)

### Модуль 5 (RAG и Agents)
- Изучите LangChain документацию
- Экспериментируйте с разными векторными БД
- Понимайте разницу между single-agent и multi-agent системами

### Модуль 6 (Deployment и Diffusion)
- Освойте Docker: базовые команды, Dockerfile
- Изучите FastAPI:路由, модели Pydantic
- Для диффузионных моделей начинайте с Stable Diffusion XL

## Частые ошибки

1. **Не читаем документацию** — всегда сначала читаем docs
2. **Не тестируем код** — пишем тесты и проверяем на разных данных
3. **Копируем без понимания** — разбираем каждую строку кода
4. **Откладываем вопросы** — спрашиваем сразу, не ждём
