# Benchmarks

Метрики и бенчмарки для оценки качества моделей.

## Метрики генерации

### Для текста
- **Perplexity:** мера качества языковой модели
- **BLEU:** качество машинного перевода
- **ROUGE:** качество суммаризации

### Для изображений
- **FID (Fréchet Inception Distance):** качество сгенерированных изображений
- **KID (Kernel Inception Distance):** альтернатива FID
- **IS (Inception Score):** разнообразие и качество

### Для RAG
- **RAGAS:** faithfulness, answer relevancy, context precision
- **Context Recall:** полнота контекста
