# LLM Prompts

Шаблоны промптов для различных задач.

## Zero-Shot промпты

```
Ты помощник для [роль]. Твоя задача — [задача].
Входные данные: {input}
Ответ:
```

## Few-Shot промпты

```
Пример 1:
Вход: {input1}
Выход: {output1}

Пример 2:
Вход: {input2}
Выход: {output2}

Теперь реши:
Вход: {input}
Выход:
```

## Chain-of-Thought

```
Давай подумаем шаг за шагом.
Вопрос: {question}
Шаг 1:
Шаг 2:
...
Ответ:
```

## Системные промпты

```
Ты — эксперт в области [область]. 
Отвечай кратко и по делу.
Если не знаешь ответа — скажи об этом.
```
