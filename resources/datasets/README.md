# Datasets

Датасеты для практических занятий.

## Рекомендованные датасеты

### Модуль 1: Введение в ML
- **Iris:** классификация цветов
- **Titanic:** предсказание выживших
- **Boston Housing:** регрессия цен на жилье

### Модуль 2-3: Глубокое обучение
- **MNIST/Fashion-MNIST:** классификация изображений
- **CIFAR-10:** классификация объектов

### Модуль 4: LLM
- **IMDB:** классификация тональности
- **SQuAD:** вопросно-ответные задачи
- **Glue:** бенчмарк для NLP задач

### Модуль 6: Генерация изображений
- **CelebA:** лица людей
- **COCO:** объекты в контексте

## Где искать датасеты

- [Hugging Face Datasets](https://huggingface.co/datasets)
- [Kaggle Datasets](https://www.kaggle.com/datasets)
