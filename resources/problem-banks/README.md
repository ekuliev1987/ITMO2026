# Problem Banks

Задачи и упражнения для практики.

## Задачи по модулям

### Модуль 1: Введение в ML
1. Загрузить датасет, выполнить EDA, визуализировать распределения
2. Обработать пропущенные значения разными методами
3. Создать интерактивный дашборд

### Модуль 2: Нейросети
1. Реализовать MLP с нуля на NumPy
2. Сравнить функции активации на одной задаче
3. Исследовать влияние регуляризации

### Модуль 3: GAN
1. Обучить DCGAN на своём датасете
2. Исследовать влияние латентного пространства
3. Реализовать WGAN

### Модуль 4: LLM
1. Fine-tuning BERT на своей задаче классификации
2. Создать систему вопросов-ответов
3. Реализовать Chain-of-Thought prompting

### Модуль 5: RAG и Agents
1. Построить RAG для своей коллекции документов
2. Создать агента с кастомными инструментами
3. Реализовать мульти-агентную систему

### Модуль 6: Deployment
1. Развернуть модель на Hugging Face Spaces
2. Создать Docker контейнер с API
3. Настроить мониторинг модели
