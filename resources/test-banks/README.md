# Test Banks

Тестовые задания для самопроверки и контроля.

## Формат тест-банков

Тест-банк может быть в любом виде, который можно использовать:
- Markdown файлы (как здесь)
- JSON / YAML
- CSV / Excel
- Специализированные форматы (LMS, Moodle и т.д.)
- Другой формат — с инструкцией по использованию

## Количество вопросов

Количество тестов для каждой лекции определяется преподавателем исходя из:
- Объёма материала
- Требуемого уровня проверки компетенций
- Формата контроля
