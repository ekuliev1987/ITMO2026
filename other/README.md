# Другие материалы

В этой папке хранятся архивные и вспомогательные материалы:

- Исходные версии РПД (docx, pdf, html)
- Презентации в различных форматах
- Скрипты для конвертации и вспомогательные утилиты
- Промежуточные версии материалов

## Структура

| Файл | Описание |
|------|----------|
| РПД - Генеративный ИИ и МО.* | Рабочая программа дисциплины в различных форматах |
| Генеративные ИИ и МО (уровневые ФОС).docx | Фонд оценочных средств |
| РПД_Главный_раздел_таблица_КРМ.* | Главный раздел РПД в логике КРМ |
| convert_to_pdf.py | Скрипт для конвертации в PDF |

**Примечание:** Эти файлы являются справочными. Актуальные материалы находятся в соответствующих разделах репозитория.
