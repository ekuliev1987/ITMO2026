# Экзамен

## Общие сведения

**Форма:** устный экзамен (2 теоретических вопроса)

**Охват:** все модули дисциплины

**Проверяемые компетенции:** ML-2, ML-3, PL-1, LLM-5, FC-2, SS-1

## Критерии оценивания

| Баллы | Критерий |
|-------|----------|
| 35-40 | Полный самостоятельный верный ответ, концептуальное понимание, примеры |
| 29-34 | Достаточно полный ответ, небольшие неточности |
| 22-28 | Неполный ответ, трудности при обобщении |
| 0-21 | Разрозненные знания, ошибки в определениях |

## Темы вопросов

### Модуль 1: Введение в машинное обучение и генеративный ИИ
1. Типы машинного обучения: с учителем, без учителя, с подкреплением
2. Дискриминативные vs генеративные модели
3. Основы работы с данными в Python (NumPy, Pandas)

### Модуль 2: Автоэнкодеры и вариационные автоэнкодеры (VAE)
4. Архитектура автоэнкодера: encoder (кодировщик), bottleneck (узкое место), decoder (декодировщик)
5. Вариационные автоэнкодеры: параметризация распределения
6. Репараметризационный трюк в VAE
7. Функция потерь VAE: reconstruction (восстановление) + KL-дивергенция

### Модуль 3: Генеративно-состязательные сети (GAN)
8. Архитектура GAN: Generator (генератор) и Discriminator (дискриминатор)
9. Минимаксная игра в GAN
10. Проблемы обучения GAN: vanishing gradients (затухающие градиенты), mode collapse (коллапс мод)
11. DCGAN и техники стабилизации

### Модуль 4: Трансформеры и большие языковые модели (LLM)
12. Механизм внимания (Attention) в трансформерах
13. Self-Attention (самовнимание) и его вычисление
14. Архитектура Transformer (Трансформер): Encoder (кодировщик) + Decoder (декодировщик)
15. Инженерия промптов (Prompt Engineering): Zero-Shot, Few-Shot, Chain-of-Thought
16. PEFT методы (Parameter-Efficient Fine-Tuning): LoRA, QLoRA
17. Архитектуры современных больших языковых моделей (LLM): Llama, Mistral, Qwen

### Модуль 5: RAG-системы и AI-агенты
18. RAG (Retrieval-Augmented Generation): архитектура и преимущества
19. Векторные базы данных: Chroma, FAISS, Pinecone
20. AI-агенты: концепция и архитектура ReAct
21. Мульти-агентные системы: LangGraph, AutoGen, CrewAI

### Модуль 6: Развёртывание и мультимодальный ИИ
22. Диффузионные модели: прямой и обратный процесс
23. Stable Diffusion: text encoder (текстовый кодировщик) + U-Net + VAE
24. Развёртывание моделей: FastAPI, Docker
25. Квантование моделей (Model quantization): INT8, INT4, GGUF
26. Мультимодальные модели: CLIP, LLaVA, GPT-4V
27. Этические аспекты генеративного ИИ: deepfakes (глубокие подделки), bias (смещения), авторское право
