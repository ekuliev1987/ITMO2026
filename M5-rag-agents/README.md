# Модуль 5. RAG-системы и мультимодальный ИИ

**Объём:** 18 часов | **Компетенции:** LLM-5, FC-2, PL-1, SS-1 | **Уровень:** Базовый/Средний

## Содержание

### Занятие 5.1. RAG-системы (Лекция, 2 ч)
- RAG (Retrieval-Augmented Generation): архитектура и преимущества
- Векторные базы данных: Chroma, FAISS, Pinecone, Weaviate
- Эмбеддинги и их роль
- Гибридный поиск

### Занятие 5.2. Построение RAG-системы (Практика, 6 ч)
- Построение RAG-системы с LangChain
- Загрузка документов, чанкинг, токенизация
- Создание векторной БД
- Retrieval + Generation пайплайн
- Тестирование на вопросах

### Занятие 5.3. Продвинутый RAG (Практика, 4 ч)
- Гибридный поиск, reranking компонентов
- Оценка качества с RAGAS метриками (faithfulness, answer relevancy, context precision)
- Оптимизация параметров

### Занятие 5.4. Мультимодальные модели (Лекция, 2 ч)
- Мультимодальные модели (CLIP, LLaVA, GPT-4V)
- Архитектурные подходы (энкодеры + выравнивание)
- Visual Question Answering
- Кейсы применения и злоупотреблений (deepfakes, плагиат, bias)

### Занятие 5.5. Мультимодальные приложения (Практика, 4 ч)
- Работа с мультимодальными API
- Visual Question Answering на практике
- Генерация изображений по текстовому описанию
- Инструменты обнаружения синтетического контента
- Этический аудит мультимодальных систем

## Компетенции КРМ

|-------------|-----------|---------|------------|
| LLM-5 | LLM-5.1, 5.5, 5.6, 5.7 | Базовый/Средний | Может строить RAG-системы, работать с мультимодальными моделями |
| FC-2 | FC-2.3 | Средний | Может решать комплексные задачи с помощью RAG и мультимодальных моделей |

## Практические задания

- Задание 5.1: Построить RAG-систему с LangChain
- Задание 5.2: Реализовать гибридный поиск и оценку RAGAS
- Задание 5.3: Visual Question Answering с мультимодальной моделью
- Задание 5.4: Этический аудит мультимодальной системы
