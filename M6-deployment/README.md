# Модуль 6. AI Agents, развёртывание и диффузионные модели

**Объём:** 26 часов | **Компетенции:** LLM-5, DL-2, PL-1 | **Уровень:** Базовый/Средний

## Содержание

### Занятие 6.1. Диффузионные модели (Лекция, 2 ч)
- Прямой и обратный процесс диффузии
- Forward diffusion (добавление шума)
- Reverse process (denoising с U-Net)
- Преимущества диффузионных моделей над GAN
- Latent Diffusion Models (Stable Diffusion)

### Занятие 6.2. Stable Diffusion (Практика, 4 ч)
- Установка diffusers
- Пайплайн Stable Diffusion (text encoder + U-Net + VAE)
- Генерация изображений по промптам
- Анализ влияния промпта на результат
- Артефакты и ограничения

### Занятие 6.3. LoRA fine-tuning для изображений (Практика, 4 ч)
- Подготовка кастомного датасета изображений + описаний
- Настройка пайплайна
- LoRA fine-tuning на кастомных данных (DreamBooth)
- Сравнение с базовой моделью

### Занятие 6.4. AI Agents: концепция (Лекция, 2 ч)
- AI Agents: концепция, отличие от чат-ботов
- Архитектура ReAct (Reason + Act)
- Типы инструментов (tools), function calling
- Мульти-агентные системы
- Фреймворки: LangGraph, AutoGen, CrewAI

### Занятие 6.5. Создание AI-агента (Практика, 4 ч)
- Создание простого AI-агента с LangChain
- Определение инструментов (поиск, калькулятор, API)
- Реализация agent loop
- Тестирование на multi-step задачах

### Занятие 6.6. Развёртывание моделей (Лекция, 2 ч)
- Этапы развёртывания в production
- Model quantization (INT8, INT4, GGUF)
- Оптимизация инференса: batching, caching, distillation
- Платформы: vLLM, TGI, Hugging Face Spaces

### Занятие 6.7. FastAPI и Docker (Практика, 4 ч)
- Создание FastAPI приложения для генеративной модели
- REST API endpoints
- Dockerfile для контейнеризации
- Развёртывание на облачной платформе
- Тестирование извне

### Занятие 6.8. Оптимизация и мониторинг (Практика, 2 ч)
- Кэширование (Redis), batch inference
- Настройка мониторинга (Prometheus, Grafana)
- Load testing
