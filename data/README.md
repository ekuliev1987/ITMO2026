## КРМ версия 3.0

Компетентностно-ролевая модель в области искусственного интеллекта определяет:

### Профессиональные роли

- **ML Engineer:** инженер по машинному обучению
- **AI Engineer:** инженер по искусственному интеллекту
- **AI Architect:** архитектор ИИ-систем
- **ML Researcher:** исследователь в области ML
- **Prompt Engineer:** инженер по промптам

### Компетенции

| Код | Название | Описание |
|-----|----------|----------|
| ML-2 | Машинное обучение | Применение фундаментальных принципов и методов ML |
| ML-3 | Глубокое обучение | Использование архитектур глубоких нейронных сетей |
| PL-1 | Программирование | Разработка программного обеспечения для ML/AI |
| DL-2 | Глубокое обучение | Архитектуры глубокого обучения |
| LLM-5 | Большие языковые модели | Работа с большими языковыми моделями |
| FC-2 | Применение ИИ | Решение практических задач с помощью ИИ |
| SS-1 | Социальные аспекты | Понимание социальных последствий ИИ |
