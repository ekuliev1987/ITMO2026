# Проектная работа

## Описание

Интегральная проектная работа, проверяющая способность применять знания из нескольких модулей для решения комплексной задачи.

## Формат

- **Форма:** групповая работа (2-4 человека)
- **Срок:** 4-6 недель
- **Результат:** работающий прототип + презентация + отчёт

## Темы проектов

### Вариант 1: RAG-система для предметной области
- Выбор предметной области и сбор датасета документов
- Построение RAG-системы с векторной базой данных
- Реализация интерфейса (веб-приложение или API)
- Оценка качества с RAGAS метриками

### Вариант 2: AI Agent для автоматизации задачи
- Определение задачи и необходимых инструментов
- Реализация агента с LangChain/AutoGen
- Тестирование на наборе задач
- Сравнение с базовыми подходами

### Вариант 3: Fine-tuning LLM для специализированной задачи
- Сбор и подготовка датасета
- LoRA/QLoRA дообучение модели
- Оценка качества и сравнение с базовой моделью
- Развёртывание модели

### Вариант 4: Мультимодальная система
- Интеграция текстовой и визуальной модальностей
- Реализация VQA или генерации изображений по описанию
- Этический аудит системы

## Критерии оценивания

| Критерий | Вес |
|----------|-----|
| Техническая реализация | 40% |
| Качество результата | 25% |
| Презентация и защита | 20% |
| Отчёт и документация | 15% |

## Требования к отчёту

1. Введение: постановка задачи, актуальность
2. Обзор подходов: анализ существующих решений
3. Методология: описание выбранного подхода
4. Реализация: архитектура, технологии, ключевые решения
5. Результаты: метрики, сравнения, анализ
6. Заключение: выводы, перспективы
7. Приложение: код, ссылки на репозиторий
