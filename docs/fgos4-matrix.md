# Матрица соответствия ФГОС 4 и Компетентностно-ролевой модели (КРМ)

> Сопоставление компетенций ФГОС ВО 4 поколения с компетенциями курса «Генеративный ИИ» для направлений бакалавриата.

---

## Общие сведения

| Параметр | Значение |
|----------|----------|
| Дисциплина | Генеративный искусственный интеллект |
| Объём | 5 ЗЕ (180 часов) |
| Семестр | 5 семестр, 3 курс |
| Компетентностно-ролевая модель | КРМ 3.0 (ML Engineer, AI Engineer, AI Architect) |

---

## Примечания

1. **УК (Универсальные компетенции)** — одинаковые для всех направлений подготовки
2. **ОПК (Общепрофессиональные компетенции)** — частично пересекаются между направлениями
3. **ПК (Профессиональные компетенции)** — специфичны для каждого направления
4. **КРМ** — компетентностно-ролевая модель в области ИИ (версия 3.0)
5. Матрица составлена на основе ФГОС ВО 4 поколения (приказы Минобрнауки 2023-2024 гг.)
