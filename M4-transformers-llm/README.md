# Модуль 4. Трансформеры и большие языковые модели (LLM)

**Объём:** 28 часов | **Компетенции:** LLM-5, FC-2 | **Уровень:** Базовый/Средний

## Содержание

### Занятие 4.1. Архитектура Transformer (Лекция, 2 ч)
- Ограничения RNN
- Механизм внимания (Attention)
- Self-Attention и его вычисление
- Архитектура Transformer (Encoder + Decoder)
- Позиционное кодирование
- Авторегрессионная генерация (GPT)

### Занятие 4.2. Hugging Face Transformers (Практика, 4 ч)
- Загрузка предобученных моделей и токенизаторов
- Masked Language Modeling (BERT)
- Генерация текста (GPT-2 / DistilGPT-2)
- Сравнение стратегий генерации (greedy, sampling, temperature, top_p)

### Занятие 4.3. Prompt Engineering (Практика, 4 ч)
- Zero-Shot, Few-Shot, Chain-of-Thought
- Итеративная отладка промптов
- Fine-tuning BERT на задаче классификации (IMDB)
- NER с помощью модели
- Оценка качества (F1-score)

### Занятие 4.4. Работа с API LLM (Практика, 2 ч)
- Работа с API современных LLM
- Системные промпты
- Управление креативностью (temperature, top_p, max_tokens)
- Сравнительные эксперименты

### Занятие 4.5. Современные архитектуры LLM (Лекция, 2 ч)
- Архитектуры: Llama 2/3, Mistral, Qwen
- Контекстное окно и его значение
- PEFT методы: LoRA, QLoRA, Prefix Tuning, Prompt Tuning
- Когда применять PEFT vs full fine-tuning

### Занятие 4.6. LoRA дообучение (Практика, 8 ч)
- LoRA дообучение с библиотекой peft
- Настройка LoRA config (r, alpha, target_modules)
- Подготовка датасета
- Обучение Llama 2 / Mistral
- Сохранение и загрузка LoRA-весов

### Занятие 4.7. QLoRA (Практика, 4 ч)
- QLoRA с 4-битной квантовацией
- Дообучение большой модели на ограниченном GPU
- Сравнение с полным fine-tuning

## Компетенции КРМ
